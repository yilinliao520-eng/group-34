# Group 34 ACL Overleaf Package

This folder is prepared from the official ACL LaTeX template downloaded from Overleaf.

Upload the whole folder contents to Overleaf:

- `main.tex`
- `acl.sty`
- `acl_natbib.bst`
- `figures/workflow.pdf`
- `figures/main_results.pdf`

In Overleaf:

1. Set **Compiler** to **XeLaTeX**.
2. Set **Main document** to `main.tex`.
3. Click **Recompile**.
4. Download the generated PDF and submit it to Elearning.

Do not use pdfLaTeX because the report body is in Chinese.
